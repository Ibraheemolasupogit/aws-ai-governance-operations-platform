# Architecture

## Overview

The platform is planned as a local-first Python application that models the governance and operations layer for AWS-hosted ML and GenAI systems.

## Current Milestone

Milestone 1 provides repository structure, configuration placeholders, documentation, and tests only. No platform business logic or AWS integration exists yet.

## Planned Components

- Inventory and model catalogue
- Policy checks
- Risk scoring
- Access review
- Audit event summaries
- Monitoring and cost summaries
- Incident and risk register
- Model cards and evidence reporting
